# AnimeSync 🎌
> リアタイのアニメ体験を未来に残すアニメ手帳。

## 一文定義
AnimeSyncは、**リアタイのアニメ体験を未来に残すアニメ手帳**。

---

## なぜ作るのか

アニメ視聴インフラは進化したが、ユーザーの「管理体験」は進化していない。

### 3つの分断

**① 情報の分断**
- 地上波最速、ABEMA先行、VOD遅延など複雑化
- 自分が契約しているサブスクの最速がどこか分からない
- 作品ごとに公式サイトを確認するしかない

**② 時間の分断**
- 深夜帯・曜日分散で管理負荷が増大
- 今週何を観るのか整理が面倒

**③ 感情の分断**
- SNSはネタバレ・ノイズが多い
- 視聴直後の「熱」を置く場所がない

---

## コンセプト：3つのSync

| Sync | 内容 |
|---|---|
| **Info Sync** | 加入サブスクに合わせた「あなたにとっての最速」だけを表示 |
| **Time Sync** | 今夜の放送・今週の運行表を整理。週次LINE通知 |
| **Emotion Sync** | 視聴後に「神 / 余韻 / 不憫」を記録。感情ログとして蓄積 |

## 使用予定API

### Annict（アニクト）
https://annict.com/

日本発のアニメ記録サービス。GraphQL APIが無料公開されている。

AnimeSyncでの活用：
- 今期放映中の作品一覧
- キャラクター情報（主人公・ヒロイン等）
- スタジオ・スタッフ情報
- ユーザーの視聴記録との連携

### しょぼいカレンダー
https://cal.syoboi.jp/

国内最大級のアニメ放送スケジュールDB。APIが公開されている。

AnimeSyncでの活用：
- 放送曜日・時間の取得
- 配信プラットフォーム別の最速情報
- 地上波・配信の放送時刻差分

### 連携イメージ

```
しょぼいカレンダー → いつ・どこで放映されるか
        ↓
Annict           → その作品・キャラ・スタジオ情報
        ↓
AnimeSync        → あなたの加入サブスクで最速はどこか
```

この2つのAPIを統合することで、
「自分にとっての最速」を自動で表示するInfo Syncが実現する。

*「深夜の司令室」——自分だけのアニメ時間を、自分でコントロールする場所。この体験をすべての設計判断の基準にする。*
